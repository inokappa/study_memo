# recipe の書き方復習

## not_if と only_if の使い方

### not_if 従来

```
execute "create file hoge" do
  command "touch /tmp/hoge"
  not_if (ls /tmp/hoge)
end
```

一応、これでも動く。

### not_if こう書いた方がカッコイイと思う（当社比）

```
execute "create file hoge" do
  command "touch /tmp/hoge"
  not_if {File.exists?("/tmp/hoge")}
end
```

上記のように書いた方がより ruby 的だし、なんとなくカッコイイ。
